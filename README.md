# Cosmic Explorer

A high-graphic, interactive space website with a built-in survival game. Built with React, Vite, and HTML5 Canvas.

## Features
- **Interactive Starfield**: Mouse-reactive background.
- **Embedded Game**: Avoid asteroids, collect stardust, and survive the void.
- **Modern UI**: Glassmorphism, neon aesthetics, and Framer Motion animations.

## Setup & Run

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Start Development Server**
    ```bash
    npm run dev
    ```

3.  **Build for Production**
    ```bash
    npm run build
    ```

## Controls
- **Movement**: Use **WASD** or **Arrow Keys** to move.
- **Controls**: Press keys to navigate your ship.
- **Objective**: Dodge purple asteroids.
